Downloads

Please Note: Due to changes that have taken place in both Java, and in its licensing, since ChairGun was last released by its Original Developer, only the “Mac - With Java JRE” bundle includes an installer to automate the installation process for you.

You must follow the instructions included with each bundle to get it to run on your device. These instructions should be well within the capability of most users.

If you follow the instructions, you should have no trouble running ChairGun USE on your device.

Enjoy!

Mac - With Java JRE

This is the easiest possible way to install ChairGun-USE on MacOS.

This is for people who don't mind a larger download, and are not overly concerned with having the latest possible Java security updates, but just want to get up and running as quickly and easily as possible with ChairGun-USE.

ChairGun-USE-Mac-App-WithJRE.zip

Size: 70.2 MB

Mac - No Java JRE

This is for advanced users who want a much smaller download, or the flexibility to determine the exact Java Runtime Environment that you want to use to run ChairGun-USE, enabling you to get the most recent Java security updates.

ChairGun-USE-Mac-App-NoJRE.zip

Size: 12.1 MB

ALL — Windows, Linux or MacOS

This is for Windows or Linux users. There is no fancy installer at this time. It requires you to follow specific instructions to install it; but it should be well within the capabilities of your average Windows or Linux user.

This will also give you the bare essentials to run ChairGun-USE on a Mac; though it is highly recommended you use either of the .app Mac packages rather than this.

ChairGun-USE-ALL.zip

Size: 11.1 MB
